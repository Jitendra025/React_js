1. Go to node,js site and install npm
2. Open command prompt and run command npm install creater-react-app -g (note -g here to install react globally on the machine so that create-react app can be run from from cmd though u r in any directory)
3.  go to any directory where you want to create your react project and run command-> create-react-app react-complete-guide --scripts-version 1.1.5
4. No go to (change directory in cmd) the folder of you project and run command-> npm start. This command will start your app in a browser
